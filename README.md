
## Setup and Usage
1. Clone and run the main app
   [https://github.com/row-huh/physio-therapy-web](https://github.com/row-huh/physio-therapy-web)

2. Make sure the app is running before executing the Selenium scripts
   The app runs on port 3000 by default, which the scripts expect.

3. Clone this repository and install dependencies:  
    `pip install -r requirements.txt`

4. Run the scripts in order:

   ```
   python test_part1_record_and_save.py
   python test_part2_compare_exercise.py
   ```
